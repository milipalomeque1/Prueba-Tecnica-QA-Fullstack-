# Prueba-Tecnica-QA-Fullstack-

PRUEBA AUTOMATIZADA
Este repositorio contiene una pequeña suite de pruebas end-to-end escrita con Playwright. La idea es tener ejemplos claros y fáciles de mantener usando el patrón Page Object Model (POM), para que los tests sean legibles y los selectores/acciones estén centralizados.

Qué encontrarás aquí
- Tests para la aplicación de ejemplo (Sauce Demo) en `milagrosPalomeque/form.spec.ts`.
- Objetos de página (Page Objects) en `milagrosPalomeque/pages/` — `LoginPage`, `InventoryPage` y una `BasePage`.
- Configuración de Playwright en `playwright.config.ts`.

Requisitos
- Node.js (v16+ recomendado).
- NPM (o cualquier gestor de paquetes compatible con Node).
